# Screenshots Directory

This directory contains visual documentation of the AETHOS application.

## Required Screenshots

Add the following screenshots to enhance your README:

1. **homepage.png** - Main application interface
2. **analysis_result.png** - Complete analysis results showing object detection, captioning, and OCR
3. **workflow.png** - Command interface showing slash commands
4. **architecture.png** - System architecture diagram
5. **demo.gif** - Animated demo of the complete workflow

## How to Create Screenshots

1. **Homepage UI**
   - Navigate to http://127.0.0.1:5000
   - Capture the main interface with terminal prompt

2. **Analysis Result**
   - Upload an image and run `/analyze`
   - Capture the complete results page

3. **Command Interface**
   - Show various slash commands being executed
   - Highlight the terminal-inspired design

4. **Architecture Diagram**
   - Create using draw.io, Lucidchart, or Canva
   - Follow the flow described in docs/architecture.md

5. **Demo GIF**
   - Record using OBS Studio or ScreenToGif
   - Show: Upload → Command → Results

## Tools for Screenshots

- **Windows**: Snipping Tool, OBS Studio, ScreenToGif
- **Online**: draw.io for diagrams, Canva for graphics
- **GIF Creation**: ezgif.com, ScreenToGif
